# Media Search

## When to Use
用户请求视频、影视、综艺、音乐、体育内容、晚上放松内容或世界杯相关媒体。

## Checklist
- 读取 memory 中的平台、片长、语言、题材、球队/球员和剧透偏好。
- 默认使用 media.video.search。
- 用户明确 YouTube-only 或海外视频源时使用 youtube.video.search。
- 世界杯场景优先找集锦、前瞻、解说、赛程相关公开内容。

## Boundaries
不编造比分、赛程、播放链接、平台内容或版权状态。
