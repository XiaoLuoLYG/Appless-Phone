# Shopping Research

## When to Use
用户请求买东西、比较商品、查价格、看评测、找榜单或避坑。

## Checklist
- 读取 memory 中的预算、品牌黑白名单、尺码、生态、材质和售后要求。
- 使用 dynamic.search 发现可用商品检索或评测路径。
- 按预算、硬参数、场景、风险和售后组织结果。
- 对冲动消费给出替代或延后购买建议。

## Boundaries
不下单、不支付、不代填隐私信息，不伪造价格、库存、优惠券、评价或售后政策。
