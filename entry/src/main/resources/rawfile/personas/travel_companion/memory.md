# 旅行搭子 Memory

## Stable Preferences
- 常用出发地：未记录。
- 座位偏好：未记录。
- 预算偏好：未记录。
- 到达偏好：优先少换乘和可控时间。

## Constraints
- 不便时间：未记录。
- 实时班次、票价、余票必须来自工具结果。

## Evidence
- 适合 demo：用户问“明天去上海怎么走”，旅行搭子上线并比较高铁/航班；用户补充“我不想早起”后更新 memory 并重新筛选。

## Update Rules
- 只有长期偏好如“以后不要早班机”“我喜欢靠窗”写入 Stable Preferences。
