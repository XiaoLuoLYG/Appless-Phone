# 12306 In-App Booking Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在现有 12306 高铁卡片折叠态显示真实的 `¥xxx起`，在展开态提供“去购票”，并在 App 内复用现有 WebView 完成 12306 官网登录、选车次、下单与支付流程。

**Architecture:** 继续由 `train.search` 读取 12306 同一份余票响应，在 `ToolGatewayClient` 内解析 `yp_info_new` 的最低可售价格并生成线路级官网 URL；通过现有 A2UI → HTML 快照 → 出行 Renderer 加入价格和客户端动作；`Index` 只接受当前文档中的动作且再次校验 URL，然后打开扩展后的 `CheckoutWebOverlay`。不新增工具、支付 provider、页面路由或本地订单状态。

**Tech Stack:** HarmonyOS ArkTS、现有 A2UI JSONL 数据模型、HtmlTravelHomeRenderer、ArkWeb `A2uiWebSurfaceView`、Hypium、DevEco `hvigorw`、HarmonyOS 真机 `hdc`。

## Global Constraints

- 实施前读取并遵循 `superpowers:test-driven-development`；每个行为先出现可解释的 RED，再写最小 GREEN。
- 保持 Ponytail 边界：复用现有 `CheckoutWebOverlay`，不新增购票页、provider adapter、后端接口、依赖或通用导航框架。
- 价格只能来自本次 12306 响应的 `yp_info_new`；不得估算、补默认价或额外发起 N+1 票价请求。
- 折叠态不显示席别，只在价格真实存在时显示 `¥xxx起`；展开态按钮文案固定为“去购票”。
- 12306 没有稳定的指定车次深链，因此官网 URL 只预填日期、出发站和目的地，用户在官网重新选择车次。
- 不注入或读取 12306 账号、乘车人、订单和支付信息；关闭 WebView 后不生成本地成功状态。
- 自定义 Scheme 默认阻止。只有真机实际观察且确认来自官方流程的 Scheme，才能另行加入精确白名单；本计划不猜测 Scheme。
- Hypium 结果必须读取 `entry/.test/default/intermediates/test/coverage_data/test_result.txt`，不能只看命令退出码。
- 规格文档：`docs/superpowers/specs/2026-07-21-12306-in-app-booking-design.md`。

在 `parse12306Row` 前加入以下 helpers。字段映射使用当前 12306 余票响应：商务 32、特等 25、一等 31、二等 30、高级软卧 21、软卧/一等卧 23、动卧 33、硬卧/二等卧 28、软座 24、硬座 29、无座 26。

```ts
function trainField(fields: string[], index: number): string {
  return fields.length > index ? fields[index].trim() : '';
}

function isAvailableTrainSeat(value: string): boolean {
  const text = value.trim();
  return text === '有' || /^[1-9]\d*$/.test(text);
}

function trainSeatAvailabilityForPriceCode(code: string, fields: string[]): string {
  if (code === '9') {
    return trainField(fields, 32);
  }
  if (code === 'P') {
    return trainField(fields, 25);
  }
  if (code === 'M') {
    return trainField(fields, 31);
  }
  if (code === 'O' || code === 'S') {
    return trainField(fields, 30);
  }
  if (code === '6' || code === 'A') {
    return trainField(fields, 21);
  }
  if (code === '4' || code === 'I') {
    return trainField(fields, 23);
  }
  if (code === 'F') {
    return trainField(fields, 33);
  }
  if (code === '3' || code === 'J') {
    return trainField(fields, 28);
  }
  if (code === '2') {
    return trainField(fields, 24);
  }
  if (code === '1') {
    return trainField(fields, 29);
  }
  if (code === 'W') {
    return trainField(fields, 26);
  }
  return '';
}

function lowestAvailable12306Price(fields: string[]): string {
  const ypInfoNew = trainField(fields, 39);
  if (ypInfoNew.length === 0 || ypInfoNew.length % 10 !== 0) {
    return '';
  }
  let minimumJiao = -1;
  for (let offset = 0; offset < ypInfoNew.length; offset += 10) {
    const chunk = ypInfoNew.substring(offset, offset + 10);
    const availability = trainSeatAvailabilityForPriceCode(chunk.substring(0, 1), fields);
    const priceText = chunk.substring(1, 6);
    if (!isAvailableTrainSeat(availability) || !/^\d{5}$/.test(priceText)) {
      continue;
    }
    const priceJiao = Number.parseInt(priceText, 10);
    if (priceJiao <= 0) {
      continue;
    }
    if (minimumJiao < 0 || priceJiao < minimumJiao) {
      minimumJiao = priceJiao;
    }
  }
  if (minimumJiao < 0) {
    return '';
  }
  const yuan = Math.floor(minimumJiao / 10);
  const jiao = minimumJiao % 10;
  return jiao === 0 ? yuan.toString() : yuan.toString() + '.' + jiao.toString();
}

export function trainLowestAvailablePriceForTest(row: string): string {
  return lowestAvailable12306Price(row.split('|'));
}
```

export type CheckoutWebKind = 'payment' | 'hotel' | 'train';

export function isAllowed12306BookingUrl(url: string): boolean {
  const value = url.trim();
  const authority = 'https://kyfw.12306.cn';
  const lower = value.toLowerCase();
  if (!lower.startsWith(authority)) {
    return false;
  }

function webHostForDisplay(url: string): string {
  const value = url.trim();
  const schemeEnd = value.indexOf('://');
  if (schemeEnd < 0) {
    return '';
  }
  const start = schemeEnd + 3;
  let end = value.indexOf('/', start);
  if (end < 0) {
    end = value.length;
  }
  const query = value.indexOf('?', start);
  if (query >= 0 && query < end) {
    end = query;
  }
  const fragment = value.indexOf('#', start);
  if (fragment >= 0 && fragment < end) {
    end = fragment;
  }
  return value.substring(start, end);
}
